## Завдання 2. Робота з документацією (2 бали)

### Виконані команди

```bash
man ls
help cd
man cat
man man
cp --help
mv --help
```

---

### Питання та відповіді

**1. Який ключ `ls` показує приховані файли?**

Ключ `-a` (або `--all`):

```bash
ls -a
```

Він відображає всі файли, включно з прихованими (назви яких починаються з крапки, наприклад `.bashrc`, `.config`).

![ls -a](assets/img/task2_ans1.avif)

---

**2. Який ключ у `cat` нумерує рядки?**

Ключ `-n` (або `--number`):

```bash
cat -n файл.txt
```

Він нумерує кожен рядок виводу починаючи з 1.

![cat -n](assets/img/task2_ans2.avif)

---

**3. Чим відрізняються `man` і `--help`?**

| Характеристика | `man` | `--help` |
|---|---|---|
| Повнота | Повна документація з описом, прикладами, опціями | Короткий список опцій |
| Навігація | Інтерактивна (можна гортати, шукати) | Просто виводиться у термінал |
| Доступність | Потребує встановлених man-сторінок | Вбудований у саму команду |
| Приклад | `man ls` | `ls --help` |

`man` — детальний довідник для глибокого вивчення команди, `--help` — швидка шпаргалка прямо в терміналі.

## Завдання 3. Міні-сценарій (2 бали)

**Сценарій:** Перейти в каталог, створити файл, переглянути список, перейти в інше місце, повернутись назад, переглянути документацію.

```bash
# 1. Перейти у каталог Документи
cd ~/Документи

# 2. Створити файл із вмістом (без touch)
echo "Homework is done" > task.txt

# 3. Переглянути список файлів детально
ls -l

# 4. Перейти у домашній каталог
cd

# 5. Повернутись назад у попередній каталог
cd -

# 6. Переглянути документацію команди ls
man ls
```

**Результат виконання:**

```
# ls -l
total 4
-rw-rw-r-- 1 aleax aleax 17 mar 29 16:59 task.txt

# cd -
/home/aleax/Документи
```

**Пояснення сценарію:**

- `cd ~/Документи` — перехід у каталог за відносним шляхом через `~`
- `echo "..." > task.txt` — створення файлу з текстом за допомогою перенаправлення виводу
- `ls -l` — перегляд списку файлів у довгому форматі (права доступу, розмір, дата)
- `cd` — без аргументів повертає у домашній каталог
- `cd -` — повертає у **попередній** каталог (зберігається у змінній `$OLDPWD`)
- `man ls` — перегляд повної документації команди `ls`
